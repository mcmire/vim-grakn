# vim-grakn

This project provides Vim support for [Grakn], specifically syntax highlighting
for Graql (.gql) files.

[Grakn]: https://grakn.ai/

## Installing

Install using your favorite package manager, or use Vim's built-in package
support (see `:help packages`).
